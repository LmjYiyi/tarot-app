# Real Data Policy

This package follows a strict rule: no generated item is presented as data.

Allowed as data:

1. Public-domain source text, with source URL and locator.
2. Open-source or open-license datasets, with license and source URL.
3. User-consented logs or commissioned expert annotations, if added later.

Not allowed as data:

1. AI-generated examples masquerading as expert samples.
2. Copied commercial website meanings without permission.
3. Synthetic question banks unless marked as synthetic and excluded from `real_data` manifests.

For commercial product use, check jurisdiction-specific copyright status, especially for RWS images outside the US and for any modern recolored editions.
